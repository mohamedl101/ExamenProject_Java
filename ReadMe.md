# Euromoon - CLI Trein Ticket Systeem

## Projectomschrijving

Dit project is een Proof of Concept Java CLI applicatie voor het fictief bedrijf "Euromoon", 
die internationale rijzen binnen Europa faciliteert.

Dit project werd uitgevoerd voor het vak "Programming Advanced" aan de Erasmushogeschool Brussel

## Functionaliteiten
1. Registratie Passagieren
2. Aanmaken van reizen
3. Trein koppelen aan reizen
4. Tickets verkopen aan passagiers (1e of 2e klasse)
5. Afdrukken boardinglijst

#### In kader van het vak focus het project zich op:

- Objectgeoriënteerd ontwerp
- Duidelijke documentatie
- Planmatig te werk gaan met een product backlog
- Foutafhandeling en validatie

## Gebruikte Technologieën

Programmeertaal: Java; Command Line Interface (CLI)

Opslag: Memory (Geen database)

## Vereisten en Gebruik

JDK 17+ aanbevolen

CLI (Command Line Interface)

Javadoc aanwezig voor uitleg klassen en methodes
